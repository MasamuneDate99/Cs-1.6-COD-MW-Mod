Compiler readme :

Put the source code ( .sma ) into one folder with the compile.exe.
After that, just run the compile.exe, and the a "compiled" folder
should appear. The folder contain the compiled plugin.

If there are some error, it might be you are missing some include.
Just fill the missing include / native in the "include" folder. 

Any new include / native must be put on the "include" folders.
